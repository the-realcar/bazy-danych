Mój projekt bazy danych dotyczy prywatnego przedsiębiorstwa usług transportowych, które posiada wiele zajezdni i rozbudowaną flotę pojazdów. Baza umożliwia zarządzanie taborem, pracownikami, przeglądami, naprawami, przydziałami oraz przetargami.

Planuję rozwinąć projekt o panel kierowcy, w którym dyspozytorzy będą mogli przydzielać brygady kierowcom, a kierowcy – sprawdzać swój grafik, składać wnioski (np. o kurs z wolnego czy przydział stałego pojazdu) i przeglądać tabor. Zarząd zyska możliwość zarządzania pojazdami, pracownikami, zajezdniami oraz wprowadzania zmian do bazy przez panel administracyjny. Dodatkowo powstanie strona dla pasażerów, gdzie będzie można sprawdzić rozkłady jazdy, trasy linii oraz obejrzeć tabor i zdjęcia pojazdów.

Projekt planuję zrealizować w technologiach HTML, CSS, JavaScript, PHP i Python.
